## 2. 시드 생성기 확장 (etf-buzz-monitor DB 연동)

### 배경
MiroFish 시뮬레이션 품질을 높이기 위해 etf-buzz-monitor의 Supabase DB에서 더 많은 데이터를 시드 문서에 포함시킴.

### 추가된 함수

| 함수 | DB 테이블 | 시드 문서 섹션 |
|------|-----------|---------------|
| `_get_db_conn()` | — | DB 연결 재사용 헬퍼 |
| `_build_ilike_params()` | — | SQL 인젝션 방지 파라미터 바인딩 |
| `get_etf_launch_cases()` | `etf_performance` + `etf_aum` + `buzz_daily` + `etf_investor_trading` | Part 4: 과거 유사 ETF 런칭 사례 |
| `get_youtube_channel_influence()` | `youtube_channel_demographics` + `youtube_videos` | Part 5: 유튜브 채널 영향력 |
| `get_competing_etf_details()` | `etf_registry` + `etf_fee` + `etf_aum` + `etf_holdings` | Part 6: 경쟁 ETF 상세 비교 |
| `get_community_reactions()` | `naver_cafe_posts` + `toss_community_posts` + `naver_blog_posts` | Part 7: 투자자 커뮤니티 실제 반응 |

## 3. ACE AI반도체TOP3+ vs TIGER 반도체TOP10 광고전략 시뮬레이션 프롬프트

### 목적
ACE AI반도체TOP3+(469150)의 개인투자자 순매수가 TIGER 반도체TOP10(396500) 대비 크게 열세인 상황에서, 어떤 광고 전략이 격차를 역전할 수 있는지 MiroFish로 시뮬레이션.

### 프롬프트 구성

**4개 시나리오 (월 5억 동일 예산):**
- A: 유튜브 올인 — 대형+중형 유튜버 협찬 + 숏폼 + 프리롤
- B: 옥외+디지털 복합 — 전광판 + MTS 배너 + 검색광고 + 유튜버 + 커뮤니티
- C: 차별화 교육 콘텐츠 — 비교 시리즈 + PB 세미나 + SEO + AMA + 연금 특화
- D: TIGER 직접 비교 공세 — 수익률 비교 숏폼 + 비교 리뷰 + 토론 유도

**5개 투자자 유형:**
- 2030 테크 투자자, 3040 연금 투자자, 4050 자산가, 국내주식 선호파, 해외주식 선호파

**10개 추적 지표:**
1. 개인 순매수 변화
2. 투자자 유형별 전환 경로
3. 버즈 점수 변화
4. 감성 변화
5. TIGER 기존 보유자 반응
6. 부정 여론 리스크
7. 경쟁사 대응
8. PB/기관 채널 영향
9. 비용 대비 효율
10. 최적 조합 제안

### 사용법

```bash
# 1. 시드 문서 생성
python generate_new_etf_seed.py --theme "AI반도체" \
  --etf-name "ACE AI반도체TOP3+" \
  --holdings "엔비디아,TSMC,브로드컴" \
  --keywords "AI반도체,엔비디아ETF,반도체ETF,TIGER반도체,ACE반도체"

# 2. MiroFish에서 시드 문서 업로드 + 프롬프트 입력
# 프롬프트 파일: prompts/ace_ai반도체_vs_tiger_광고전략.txt
```
