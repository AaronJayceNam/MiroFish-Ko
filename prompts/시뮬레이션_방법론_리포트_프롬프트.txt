아래 시뮬레이션의 방법론과 상세 내역을 설명하는 리포트를 만들어줘. 이 리포트를 읽는 사람이 "이 시뮬레이션이 어떻게 돌아간 건지" 궁금할 때 모든 의문을 해소할 수 있어야 해.

리포트에 반드시 포함해야 할 내용:

1. 시뮬레이션 시스템 설명
- MiroFish가 무엇이고 어떤 원리로 작동하는지
- 시드 문서 → 온톨로지 → 그래프 → 에이전트 → 시뮬레이션 각 단계가 어떻게 연결되는지
- 사용된 AI 모델(GPT-5.2)의 역할과 특성

2. 에이전트 전체 목록과 설정
- 48개 에이전트의 이름, 유형(투자자/인플루언서/증권업계/운용사/ETF/기타), 주요 설정
- 13명 투자자 에이전트 각각의 나이, 직업, 투자 성향, 정보 소비 채널, 보유 종목, ETF에 대한 초기 태도
- 4명 인플루언서 에이전트의 채널 규모, 콘텐츠 스타일, 협찬 여부
- 2명 증권업계 에이전트의 역할과 추천 기준
- 4개 운용사 에이전트의 경쟁 전략

3. 10가지 투자자 유형 분류 체계
- 각 유형의 정의, 투자 계좌, 정보 소비 채널, 매수 결정 방식
- 각 유형에 해당하는 에이전트가 누구인지
- 이 유형 분류가 어떤 리서치(한국 리테일 투자자 ETF 정보 탐색 경로 조사)에 기반하는지

4. 9개 시나리오 상세
- 시나리오 A~I 각각의 내러티브, 핵심 메시지, 타겟 투자자 유형
- 마케팅 시나리오(A~F)와 시장 환경 시나리오(G~I)의 차이
- 각 시나리오가 ACE K수출 ETF에 유리한지/불리한지

5. 시뮬레이션 조건과 설정
- 시뮬레이션 기간(12주), 라운드 수, 플랫폼(트위터+레딧)
- 총 액션 수, 플랫폼별 액션 수, 액션 유형별 분포(게시글/댓글/좋아요/리포스트/인용/아무것도안함)
- 그래프 규모(노드 수, 엣지 수), 엔터티 타입, 관계 타입

6. 12주 이벤트 타임라인
- 각 주차에 주입된 이벤트 (ETF 상장, 수출 통계, 경쟁사 보수 인하, 삼양식품 실적, 핀플루언서 규제, 환율, 코스닥 돌파, 토스 추천, 코스피 돌파, 미국 시장 랠리 등)
- 각 이벤트가 어떤 시나리오를 테스트하기 위해 설계되었는지

7. 에이전트별 활동 내역
- 48개 에이전트 각각의 총 활동 횟수
- 활동량 상위 20개 에이전트의 게시글/댓글/좋아요/리포스트 분포
- 투자자 에이전트 13명 각각의 실제 발화 샘플(트위터/레딧에서 쓴 글)
- 인플루언서 에이전트의 실제 콘텐츠 샘플

8. 에이전트 간 상호작용 패턴
- 누가 누구에게 영향을 미쳤는지 (예: 투자왕김선생 → 박준혁, 머니톡TV → 이서연)
- 트위터와 레딧에서의 토론 구조 (어떤 주제로 토론이 형성되었는지)

9. 비용 분석
- 탐색 시뮬레이션(gpt-4o-mini) 비용
- 최종 시뮬레이션(GPT-5.2) 비용
- 총 비용과 단계별 토큰 사용량 추정

10. 한계점과 개선 방향
- 이 시뮬레이션의 한계 (매수 체결 로그 부재, 인터뷰 오류 등)
- 다음 시뮬레이션에서 개선할 점
- 시드 문서를 더 보강하면 어떤 효과가 기대되는지

형식: HTML, 바탕화면에 저장, 모든 데이터를 표와 차트로 시각화, 쉬운 한국어로 작성
